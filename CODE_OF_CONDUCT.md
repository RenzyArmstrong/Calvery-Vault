# Code of Conduct

Kami mengadopsi [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Singkatnya:

- Ramah dan inklusif
- Hargai perbedaan pendapat
- Terima feedback dengan anggun
- Fokus ke yang terbaik untuk komunitas
- Empati pada anggota komunitas lain

Pelanggaran bisa dilaporkan ke `conduct@calvery.xyz`.
