# Melaporkan Bug & Request Fitur

Calvery Vault (CVSM) adalah **distribusi binary + konfigurasi**. Source
code tidak dipublikasikan, jadi pull request dari komunitas tidak diterima.
Tapi **laporan bug, request fitur, dan feedback tetap sangat welcome** —
tim Calvery yang akan memperbaiki dan merilis versi baru.

## Bug report

Buka di [GitHub Issues](https://github.com/RenzyArmstrong/Calvery-Vault/issues/new?template=bug_report.md) dengan template yang tersedia.

Minimal sertakan:

- Deploy method (Docker Compose / systemd / Kubernetes)
- CVSM version (`cvsm --version` atau tag image yang dipakai)
- OS + arsitektur
- Langkah reproduksi
- Error message / log (redact secret kalau ada)

## Feature request

Buka [GitHub Issue](https://github.com/RenzyArmstrong/Calvery-Vault/issues/new?template=feature_request.md) → pilih template **Feature request**.

Fitur yang memiliki use case kuat + vote komunitas banyak akan diprioritaskan
di roadmap.

## Security vulnerability

**Jangan** buka issue publik. Laporkan privat via:

- [GitHub Security Advisory](https://github.com/RenzyArmstrong/Calvery-Vault/security/advisories/new) (preferred)
- Email: security@calvery.xyz

Detail lengkap di [SECURITY.md](SECURITY.md).
